TrumpGame

Par Alexandre Piguet le 23/03/2020 à 09:00

Description : Donald Trump se retrouve seul face à une armée de zombies mexicains . Il doit construire un mur de la Paix afin de s'en protéger.

Notre hero dispose d'armes à ramasser dispersées dans le monde. Il devra également récolter des briques pour son mur et vaincre une bonne fois pour toutes les Mexicains.

Pour cela, il aura un long chemin à suivre et devra survivre. Il disposera d'un nombre de vie, qui seront enlevées à chaque fois qu'un méchant zombie le touchera.

Le code source de l'application est disponible sur le github du projet.
